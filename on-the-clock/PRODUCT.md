# 薪水小偷 / On The Clock - Product Brief

## Product Overview

**Name**: 薪水小偷 / On The Clock  
**Type**: Time-tracking web app (humor + productivity)  
**Register**: product (UI serves the core function)

## Users

- **Primary**: Office workers aged 22–40 (Taiwan/UK)
- **Context**: Using during work breaks, sharing results after work
- **Motivation**: Quantify procrastination time, self-mockery, compete with friends
- **Language**: Traditional Chinese / English
- **Currency**: NT$ / £ / $ (selectable)

## Core Function

User sets salary → starts timer during breaks → app calculates "stolen money" → displays as fun metric → share results

## Tone & Brand

- **Tone**: Self-mocking, playful, shareable
- **Vibe**: NOT corporate / sleek / SaaS-like. Humor-first.
- **Visual**: [See DESIGN.md] Currently Apple style for testing (originally hand-drawn marker aesthetic)

## Key Features (v1)

1. Salary setup (hourly/monthly/yearly modes)
2. Timer (auto-start, shows real-time earnings)
3. Results display (time + money earned this session)
4. Daily history (auto-reset at midnight)
5. Multilingual & multi-currency
6. Emergency "Boss Key" (fake Outlook inbox)
7. Share button

## Success Metrics

- Functional accuracy (salary calculations correct)
- Responsive (mobile ≤520px, desktop >520px)
- Playful feel (not sterile)
- Easy to share

## Anti-references

- ❌ Corporate dashboards
- ❌ Over-designed animations
- ❌ Generic SaaS UI

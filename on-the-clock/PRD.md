# 薪水小偷 / On The Clock
## 產品需求文檔 (PRD)

---

## Project Overview

薪水小偷是一款針對上班族的計時應用程式，讓使用者在工作中記錄「摸魚時間」，並將其量化為金錢損失。核心精神是**自嘲好玩**，幫助上班族用輕鬆幽默的方式看待工作中的浪費時間，同時提供一個社群分享的平台。

使用者可以在上班時間開始計時，停止後立即看到自己「偷走」了多少薪水。應用程式每 24 小時自動重置，記錄每日摸魚成績，並支援分享到社群媒體（如 Instagram）炫耀。

**核心價值主張**：
- 將無形的「浪費時間」轉化為可視化的「金錢」
- 增加職場生活的趣味性，減輕工作壓力
- 提供社群分享的樂趣，與朋友比較摸魚成績

## User Motivations & Pain Points

### 痛點
- 工作太無聊，摸魚時間沒有量化記錄
- 不知道一天到底浪費了多少時間和薪水
- 職場生活缺乏趣味和放鬆的方式

### 核心動機
1. **好玩** — 用自嘲方式記錄摸魚，增加工作中的樂趣
2. **炫耀** — 跟朋友比較、分享摸魚成績，獲得認同感
3. **量化** — 看得到具體浪費多少時間和薪水，轉化為可理解的數字

### 使用場景
| 場景 | 時間 | 行為 |
|------|------|------|
| **記錄** | 上班中 | 開始/停止計時，每次摸魚都記錄下來 |
| **查看** | 上班中或下班後 | 檢視本次摸魚成績、每日累積金額 |
| **分享** | 下班後 | 截圖或分享到 IG，與朋友炫耀 |
| **重置** | 午夜 12:00 | 自動歸零，迎接新的一天 |

## Key Features & Milestones

### Milestone 1: Core Timer Engine（核心計時引擎）
**目標**：實現基礎的計時和薪資計算功能

- **薪資計算引擎**
  - 支援三種薪資模式：時薪 / 月薪 / 年薪
  - 自動換算每分鐘薪資
  - 使用者輸入薪資後自動持久化（localStorage）

- **計時器**
  - 視覺化碼錶（260px 紅色圓形，rough.js 手繪風格）
  - 開始/停止計時控制
  - 計時頁面進場時自動開始計時

- **成績展示**
  - 計時停止後顯示本次「偷走」的金額（紅色 64px 大字）
  - 金額實時更新（計時中）

**完成標誌**：使用者可以設定薪資、開始計時、看到金額，所有數據正確無誤

---

### Milestone 2: Daily Reset & History（每日重置與紀錄）
**目標**：實現每日自動重置和歷史記錄功能

- **每日重置**
  - 午夜 12:00 自動歸零每日成績
  - 重置邏輯基於客戶端日期

- **歷史紀錄存儲**
  - 使用 localStorage 存儲每日計時紀錄
  - 記錄內容：日期、計時次數、累積金額
  - 支援查詢當日及歷史數據

- **紀錄頁面**
  - 顯示今日戰績（黃底無框）
  - 列表展示歷史紀錄
  - 清空紀錄按鈕

**完成標誌**：使用者可以查看每日紀錄，午夜正確重置，數據持久化無誤

---

### Milestone 3: Social Features（社群分享功能）
**目標**：支援多語言、多幣別、老闆緊急鍵、分享按鈕

- **多語言支援**
  - 繁體中文 / 英文切換
  - Header 右側切換按鈕
  - 所有文案本地化

- **多幣別支援**
  - 支援 NT$ / £ / $（美金）
  - 獨立於語言選擇
  - 薪資計算自動適配幣別

- **老闆緊急鍵**
  - 底部灰色橫幅 + 紅色緊急鍵按鈕
  - 按下後顯示假 Outlook 郵件收件匣全屏
  - 計時在背景繼續進行
  - Esc 返回應用程式

- **分享按鈕**
  - 結果頁顯示分享按鈕
  - 基礎版本：分享結果文案和金額
  - 未來版本（v2）：美化的分享圖片

**完成標誌**：使用者可以切換語言和幣別，老闆鍵正常隱藏應用，分享按鈕可用

---

## Technical Constraints & Notes

- **跨平台一致性**：需確保 localStorage 在所有瀏覽器中運作一致
- **時區問題**：24 小時重置基於客戶端日期，可能在跨時區使用時有邊界狀況
- **老闆鍵隱蔽性**：計時應在老闆鍵背景繼續，不會被中斷
- **手繪風格限制**：rough.js 可能在某些解析度下顯示不一致，需測試

# 薪水小偷 / On The Clock — v0 UI 生成 Prompt

## 📋 項目概述

**應用名稱**：薪水小偷 / On The Clock  
**類型**：摸魚計時 App — 記錄上班時間浪費，量化為金錢損失  
**核心**：自嘲好玩、社群分享、輕鬆幽默的工作娛樂工具

---

## 🎨 設計系統（v4.0 手繪涂鴉黑白）

### 色彩
```
背景：#FFFFFF（純白）
前景：#000000（純黑）
重點：純黑線框邊界
零灰色、零彩色
```

### 字體
```
標題（H1/H2）：Caveat 或 Kalam
  - 粗體手寫馬克筆風格
  - 可傾斜 -2° 到 +2°（隨性感）
  - 可沿弧線排列（非死板水平）
  
正文 & 按鈕：Courier Prime 或 IBM Plex Mono
  - 打字機等寬字體
  - 行距緊密 1.3
  - 清晰工整對比標題隨性
```

### 組件樣式
```
按鈕（Button）
  - 邊框：2px 純黑
  - 背景：純白
  - 圓角：0px（方角）或最多 4px
  - 內距：12–24px
  - 高度：最少 48px（觸控友善）
  
狀態：
  - Normal：2px 黑邊 + 白內 + 黑字
  - Hover：3px 黑邊（加粗）
  - Active：黑背景 + 白字
  - Disabled：opacity 0.5（淡化）

輸入框（Input）
  - 邊框：2px 純黑
  - 背景：純白
  - 圓角：0–4px
  - 高度：44–48px
  - Focus：3px 黑邊（加粗）

卡片（Card）
  - 邊框：2px 純黑
  - 背景：純白
  - 內距：16–24px
  - 可略微傾斜 -0.5° 到 0.5°（隨性感）
```

### 排版規則
```
版面最大寬度：560px（置中）
頁面邊距（padding）：16–24px
區塊間距：24–32px（垂直）
允許不對稱散落、留白充足
```

## 📱 頁面設計清單

### 頁面 1：Setup（設定頁）

**功能**：
- 用戶輸入薪資、選擇薪資模式
- 自動計算每分鐘薪資
- 開始計時

**版面結構**：
```
┌─────────────────────────────┐
│                              │
│  先算算摸魚價值!            │  ← H1 標題（Caveat，80px，可傾斜）
│                              │
│  選擇薪資模式：              │  ← 說明文字（Courier Prime）
│  ◯ 時薪  ◯ 月薪  ◯ 年薪      │  ← 三個單選鈕（小圓點 + 文字）
│                              │
│  薪資金額                    │  ← 標籤（Caveat，24px）
│  [___________]  [選幣別▼]   │  ← 輸入框 + 幣別按鈕並排
│                              │
│  每分鐘薪資：NT$5.5          │  ← 計算結果（大黑字）
│                              │
│  [ 開始摸魚 ]                │  ← 主按鈕（黑邊白內，48px）
│                              │
└─────────────────────────────┘
```

**具體需求**：
- 標題「先算算摸魚價值!」Caveat 80px，黑色，略傾斜
- 三個薪資模式單選鈕（時薪 / 月薪 / 年薪）
- 薪資輸入框：2px 黑邊、白內、Courier Prime 字、黑字
- 幣別選擇按鈕：並排輸入框右側、寬 110px、黑邊、白內
- 「每分鐘薪資」顯示：大黑字 48px、Courier Prime
- 「開始摸魚」按鈕：2px 黑邊、白內、Caveat 字、48px 高、hover 時邊框加粗

---

### 頁面 2：Timer（計時頁）

**功能**：
- 展示碼錶倒計時（MM:SS）
- 即時顯示「偷到」的金額
- 開始 / 暫停計時
- 結束計時

**版面結構**：
```
┌─────────────────────────────┐
│                              │
│      ⊙ 00:05 ⊙              │  ← 圓形碼錶（260px）+ 時間中央
│     進度環（黑色線）          │
│                              │
│                              │
│      你已摸魚                │  ← 標籤（24px）
│      NT$27.5                │  ← 大紅數字改黑 64px Courier Prime
│                              │
│                              │
│   [ ▶ ]   [ 結束摸魚 ]      │  ← 暫停按鈕 + 結束按鈕（48px）
│                              │
└─────────────────────────────┘
```

**具體需求**：
- 碼錶：260×260px 黑色圓環（純線稿）、中央 MM:SS 黑字 36px
- 標籤「你已摸魚」：Caveat 24px、黑色
- 金額顯示：Courier Prime 64px、黑色粗體
- 按鈕組：▶（暫停）+ 「結束摸魚」，各 48px 高、黑邊白內
- 允許靠中或略傾斜

---

### 頁面 3：Result（結果頁）

**功能**：
- 展示本次摸魚成績（金額 + 時間）
- 展示今日累計

```
┌─────────────────────────────┐
│  今天摸魚成績                │  ← H1 Caveat 56px
├─────────────────────────────┤
│  [本次戰績] [今日戰績]       │  ← Tab 切換（黑邊框）
├─────────────────────────────┤
│ 【Tab 1：本次戰績】          │
│                              │
│  本次金額        NT$27.5     │  ← 標籤 + 大黑字 56px
│  時間            0分5秒      │  ← 標籤 + 黑字 36px
│                              │
│  ┌─────────────────────┐    │
│  │ 今日累計 NT$150.2   │    │  ← 黃底改成淡灰
│  └─────────────────────┘    │
│                              │
│  第 1 次  0分5秒   NT$27.5  │  ← 紀錄列表（序號灰、金額黑）
│  第 2 次  0分3秒   NT$16.5  │
│  第 3 次  0分2秒   NT$11.0  │
│                              │
│  [ 查看更多 ]                │  ← 次按鈕
│                              │
│  ┌─ 繼續摸魚 ─┬─ 分享 ──┐ │  ← 兩個主按鈕並排
│  └────────────┴────────┘ │
│  [ 重新設定薪資 ]         │  ← 次按鈕（可選）

│ ... 同樣結構，但顯示累計資訊  │
│                              │
└─────────────────────────────┘
```

**具體需求**：
- 標題「今天摸魚成績」Caveat 56px 黑色
- Tab 列：「本次戰績」｜「今日戰績」，黑邊框 2px、點擊時稍加粗
- 本次金額：Courier Prime 56px、黑色粗體
- 本次時間：Courier Prime 36px、黑色
- 今日累計框：淡灰背景（或純白 + 黑邊框），內含黑字 58px
- 紀錄列表：Courier Prime 20px、序號灰色、金額黑色、逐行分隔
- 按鈕：「繼續摸魚」主按鈕（黑邊白內）｜「分享」主按鈕、下方「重新設定薪資」次按鈕
- 按鈕高度：主 48–56px、次 44px

## 🎯 交互流程

### 完整使用流程
```
1. 首次進入 → Setup 頁
   └─ 輸入薪資 → 選擇模式 → 點「開始摸魚」

2. 進入 Timer 頁
   └─ 自動開始計時（顯示 MM:SS + 實時金額）
   └─ 點「結束摸魚」→ 進入 Result 頁

3. 進入 Result 頁
   └─ 顯示本次成績 + 今日累計
   └─ 可點「繼續摸魚」 → 回 Timer 頁
   └─ 可點「分享」 → 複製分享文案
   └─ 可點「重新設定薪資」 → 回 Setup 頁

4. 點 Header 漢堡菜單 → 四個頁面快速導航
   ├─ Setup
   ├─ Timer
   ├─ Result
   └─ History

5. 點 Back Arrow（Setup 隱藏）
   ├─ Timer → 清除計時 + 回 Setup
   ├─ Result → 回 Timer
   └─ History → 回 Result
```

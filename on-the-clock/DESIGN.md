## 🎨 色彩系統（純黑白，零妥協）

```css
:root {
  --bg: #FFFFFF;           /* 純白背景 */
  --fg: #000000;           /* 純黑前景 */
  
  /* 無灰階、無彩色 */
}

body {
  background: var(--bg);
  color: var(--fg);
}
```

**鐵則**：
- ✅ 只用 #FFF（白）和 #000（黑）
- ❌ 不用任何灰色
- ❌ 不用任何彩色
- ❌ 不用透明度製造灰感（如 rgba）

## 🖊️ 字體 Typography

### 字體選擇

| 用途 | 字體 | 特性 | 何時用 |
|------|------|------|--------|
| **Hero Title** | Caveat / Kalam / Permanent Marker | 粗體手寫馬克筆，隨性傾斜 | H1 大標題 |
| **Section Heading** | Caveat / Kalam | 手寫風，水平排列 | H2 頁面小標題 |
| **Annotation** | Architects Daughter / Space Mono | 手寫等寬工程字 | 小說明、標籤 |
| **Body** | Courier Prime / IBM Plex Mono | 打字機等寬 | 正文、列表 |

### 字級規範

| 層級 | 大小 | 字體 | 特色 | 用途 |
|------|------|------|------|------|
| **Hero Title** | 80–140px | 手寫標題 | 可任意傾斜/沿弧線排列，字母大小不一 | 頁面主標題（setupTitle / resultTitle） |
| **Section Heading** | 32–48px | 手寫標題 | 水平排列，隨性手感 | 頁面副標題 |
| **Number Label** | 28–40px | 手寫標題粗體 | 大字做視覺錨點 | 日期、計數、排行 |
| **Annotation** | 14–18px | 手寫等寬 | 字距拉寬（letter-spacing: 0.05em+） | 欄位名、小說明、標籤 |
| **Body** | 11–13px | 打字機字體 | 行距緊密（line-height: 1.3） | 正文、列表項 |
| **Button Text** | 16–20px | 手寫標題 或 打字機 | 粗體、清楚可讀 | 所有按鈕 |

### 實作重點

- ✅ Hero Title 可**傾斜 -5° 到 +15°**，不必完全水平
- ✅ Hero Title 可**沿弧線排列**（CSS `transform: rotate` 逐字操作）
- ✅ Annotation 拉大字距（`letter-spacing: 0.05em` 或更大）
- ✅ Body 行距 1.3（緊湊打字機感）

## 📐 插畫與裝飾

### 手繪線稿風格

| 特性 | 規範 |
|------|------|
| **線條** | 粗細不均（2–4px），不用平滑貝茲曲線 |
| **邊框** | 純線稿（outline only），模擬手繪筆觸 |
| **填色** | 只有少數強調元素可局部填黑，其餘純線稿 |
| **對稱性** | 刻意不完美、不對稱 |

### 涂鴉符號（Meme 風）

散落在空白處，增加趣味與情緒感：

```
？ 問號（疑惑感）
★ 星星（強調/興奮）
◇ 菱形（視覺點綴）
→ 箭頭（指引）
☆ 空心星（輕鬆感）
〰 波浪線（連接/流動）
💭 對話框/想法泡泡（評論）
★ ✓ ⊗ 等符號（確認/否定）
```

**使用原則**：
- 隨機散落，不要對齐成列
- 大小不一（8–32px）
- 只用黑色線稿（不填充）
- 靠近重要文字或邊角，製造「手貼上去」的感覺

## 📏 版面與間距

### 主視覺頁（如 Setup 首屏）

```
純白背景 + 大量留白
└─ 元素刻意不對稱散落
   ├─ 大 Hero Title（可傾斜）+ 小涂鴉
   ├─ 正文打字機字體，靠角落排列
   └─ 插畫 + 符號點綴（角落或邊界）
```

**排版原則**：
- 不必居中對齊，可偏左/偏右/斜角排列
- 文字可斜切過版面邊界
- 間距不均勻（營造隨性感）

### 內容頁（Setup/Timer/Result/History）

```
┌─────────────────────┐
│ [插畫] [副標題]      │  ← 角落放置，不突出
│   ?                 │
├─────────────────────┤
│ 正文（Courier字）   │  ← 打字機字體，行距 1.3
│ 清晰工整排列        │
├─────────────────────┤
│ 01. 重要編號        │  ← 大手寫字做錨點
│ 02. 列表項          │
└─────────────────────┘
```

**特色**：
- 插畫置於角落或邊界（低調）
- 正文用等寬打字機字體（工整對比隨性標題）
- 編號用大手寫字（視覺層級）
- 邊框用極細黑線（1px），乾淨工整

### 表格/清單類

```
┌─────┬──────────┐
│ 01  │ 項目 A   │  ← 1px 黑線框
├─────┼──────────┤
│ 02  │ 項目 B   │
└─────┴──────────┘
```

- 邊框：1px 純黑
- 內距：12–16px
- 字體：Courier Prime（打字機感）
- 與主視覺的隨性形成對比（刻意的秩序感）

## 🎯 組件設計

### 按鈕

**外觀**：
- 邊框：2–3px 純黑
- 內部：純白（無填充）
- 文字：黑色、粗體手寫字或打字機字
- 形狀：矩形（可略有傾斜，-2° 到 +2°）

**狀態**：

| 狀態 | 邊框 | 內部 | 文字 |
|------|------|------|------|
| Normal | 2px 黑 | 白 | 黑粗體 |
| Hover | 3px 黑 | 白 | 黑粗體（加下劃線可選） |
| Active | 4px 黑 | 黑填充 | 白 |
| Disabled | 2px 黑（虛線） | 淡灰 → 不行！改成純白 + 減淡效果（opacity 0.5） | 黑淡化 |

**尺寸**：
- 高度：48–56px（遊戲化友善）
- 寬度：100%（流動）或 `min(100%, 280px)`（置中時）
- 圓角：0px（矩形）或 2–4px（微圓）

### 輸入框

**外觀**：
- 邊框：2px 純黑
- 背景：純白
- 文字：黑色、打字機字體
- 形狀：銳角矩形（可略有傾斜）

**狀態**：

| 狀態 | 邊框 | 背景 | 文字 |
|------|------|------|------|
| Normal | 2px 黑 | 白 | 黑 |
| Focus | 3px 黑 | 白 | 黑 + 邊框加粗 |
| Disabled | 2px 黑（虛線） | 白 | 黑淡化 |

**尺寸**：
- 高度：44–48px
- 內距：12–16px
- 邊框：2px

### 卡片

**外觀**：
- 邊框：2px 純黑
- 背景：純白
- 內距：16–24px
- 圓角：0px（矩形） 或 2–4px

**排列**：
- 卡片間距：16–24px
- 可略微傾斜（-1° 到 +1°），增加隨性感
- 邊角可加小涂鴉符號（？★）

## 🎭 設計原則總結

1. ✅ **純黑白，零妥協** — 沒有灰色、沒有彩色
2. ✅ **隨性標題 + 工整正文** — 手寫大膽 vs 打字機精確的對比
3. ✅ **大量留白** — 不要塞滿，讓頁面「呼吸」
4. ✅ **不對稱散落** — 涂鴉符號、元素位置隨性，勿對齐
5. ✅ **誇張表情 + Meme 感** — 幽默、不完美、「手貼上去」的隨意感
6. ✅ **線稿插畫** — 手繪線條（粗細不均）、純線稿、少量填黑
7. ✅ **純線框** — 按鈕/卡片都是黑邊框白內部，乾淨清楚
